Collab Edge

    •  

  • 📖 Overview

    🔹 Collab Edge is a collaboration platform where users can create posts to find partners and connect with like-minded individuals to work on shared goals.

    🔹 Users can send and receive connection requests, approve collaborators, and manage project discussions and contact sharing—making collaboration smooth and structured.

    🔹 Key Features:
    ● Create collaboration posts with project details
    ● Send connection requests to post owners
    ● Approve or decline collaboration requests
    ● Share contact details securely after approval
    ● View and manage all active collaboration posts
    ● Real-time notifications and dashboard activity tracking


    💻 Tech stack & More

    ✅ React.js frontend with custom CSS
    ✅ Node.js + Express backend
    ✅ MongoDB for dynamic data storage
    ✅ Firebase Authentication and Cloud Messaging
    ✅ Admin dashboard with analytics
    ✅ Custom logic for profile matching and approvals
    ✅ Fully responsive, cross-device design
